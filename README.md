# Chat System Database Management System (SQL)

## Project Description

The **Chat System Database Management System** is a SQL-based database project designed to manage the core functionalities of a messaging application.

The project models a chat application similar to modern messaging platforms by managing users, group chats, messages, notifications, settings, and attachments. It demonstrates database design concepts including **tables, relationships, constraints, queries, joins, and data manipulation operations**.


## Features

### User Management
Stores and manages user information:

- User ID
- Username
- Email
- Contact
- Age
- Status message
- Last seen status
- Password information


### Group Chat Management

Manages group-related information:

- Group creation
- Group name
- Admin users
- Permissions
- Member count
- Chat lock status
- Creation date


### Message Management

Handles communication data:

- Sender information
- Receiver information
- Message description
- Read/unread status
- Timestamp
- Attachments


### Notification System

Stores user notifications:

- Notification type
- Notification content
- Notification timestamps
- User-specific notifications


### User Settings

Manages application settings:

- Chat history
- Storage and data settings
- Notification preferences
- Blocked contacts
- Theme settings
- Account information


### Attachment Management

Stores file-related information:

- File name
- File size
- File type
- Linked messages
